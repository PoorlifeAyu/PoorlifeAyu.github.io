HTML（入门基础与布局标签）（超文本标记语言）

HTML的定义

定义：使用标签进行编辑让浏览器把我们想要给用户展示的超文本按照理想的解构显示出来的语言。

静态页面和动态页面的区别(面试题)

  • 静态页面：

    • 当写完前端代码后，不做任何修改，在不同时间，空间，地点，打开浏览器，看到的状态是一样的。

  • 动态页面：

    • 当写完前端代码后，不做任何修改，在不同时间，空间，地点，打开浏览器，由于后台数据发生变化看到的状态不一样。

三个版本和一个标准 迭代—-升级

  • 三个版本

    • HTML4.0.1—–pc端—–经典
    • XHTML1.0—–语法严格
    • HTML5—–新技术—–移动端专用

  • 一个标准

    • W3C—–万维网联盟 开源

HTML的基本结构【重点】

  1. <!DOCTYPE html> -----声明（指代强调我们现在使用的文档是html做的编辑操作，浏览器按照html的编译来显示）

  2. 标签的认识：

    • 单标签：<标签名>
    • 双标签：<标签名></标签名>

  3. 书写一对html标签—–规范所有html代码的书写范围

  4. 在html中书写一对head标签—–对超文本做细节设置的内容都放在这里面

    1.<meta charset='UTF-8'>-----设置字符集为国际通用编码防止中文乱码
2.<title></title>-----修改网页的标题

  5. 书写一对body标签—–我们需要给用户展示的所有内容代码都写在body中

属性的定义和使用状态

【特性】

  1. 属性必须写在开始标签中
  2. 以属性名=’属性值’来设置细节操作
  3. 多个属性用空格隔开

语法结构：
双标签：
<开始标签 属性名1=“属性值1” 属性名2=“属性值2”></结束标签>
单标签：
<单标签 属性名1=“属性值1” 属性名2=“属性值2”>

HTML 超链接 多媒 列表

新语义标签的含义

  • 通过标签命就知道它要做什么事情的标签

a标签的两种常用链接方式和可被js替代的链接方式【重点】

  • 超链接的定义

    通过设置 一个文件的路径，点击超链接标签让浏览器跳转显示的操作

    语法结构：

             <a href="设置跳转路径">文本内容</a>

    【特点】文字没访问蓝色 访问以后是紫色 自带下划线

  • 三种连接方式—重点

  1. 外链式：网站跳网站

    • 域名发生变化 域名（以http://或者https://开头的路径）

  2. 内联式：同一个网站从一个页面跳转到另一个页面

  3. 锚点链接：利用a标签实现坐标的操作，点击跳转到同一页面的不同位置

相对路径与绝对路径的区别与使用【重点】

  • 绝对路径：从http://或者https://以及’/‘还有盘符（C:D:E:F)开头的路径都叫做绝对路径

  • 相对路径：从开始写链接标签的文件作为起点，去找要访问的重点文件的路径

    同级查找: ./目标文件名
    文件夹向下查询: ./文件夹名/目标文件名
    返回上一个层级：../目标文件名

图片标签的使用与相关属性的认识

image:<img>(单标签)   src图片路径
语法结构：---（重点）
        <img src="图片相对路径">

        图片：jpg(有背景颜色),png,gif

 相关属性认识：
            alt-----一旦图片路径加载错误出现的提示说明
            title-----图片正常加载显示后鼠标移入图片上出现的文字提示说明
            width-----设置图片大小等比例缩放（不需要单位可以是百分比）

        视频标签：video 双标签  src音频路径  controls poster

HTML表格与表单

表格的作用与相关属性

  1. 表格的定义：由很多个单元格按照行和列的排布实现的一个整体，用来展示归类的数据排列

  2. 表格的使用

    • table—-双标签—-占位实现一个表格
    • tr——-双标签—-表格的行
    • td——-双标签—-表格单元格
    语法结构：---（重点）
                3*3表格
                <table>
                    <tr>
                        <td>1-1</td>
                        <td>1-2</td>
                        <td>1-3</td>
                    </tr>
                     <tr>
                        <td>2-1</td>
                        <td>2-2</td>
                        <td>2-3</td>
                    </tr>
                     <tr>
                        <td>3-1</td>
                        <td>3-2</td>
                        <td>3-3</td>
                    </tr>
                </table>

  3. 相关属性

    border----边框属性
            cellspacing----改变单元格的间距
            cellpadding----改变文字和单元格边框距离
            width-----设置宽度
            height----设置高度
            align-----用在table上表格水平居中，用在td上文字水平居中
            bgcolor----改变背景颜色

            表格合并：
                横向合并
                colspan---横向合并
                纵向合并
                rowspan--纵向合并

            tbody---是浏览器编译代码后生成的一个表格体的标签

表单的基本结构搭建

  1. 表单的定义与作用：表单就是我们使用一些输入框和提交按钮来实现用户输入数据的采集以及发送的标签结构

  2. 表单的基本创建

    • form—–双标签—表单标签—一般写在最外层
    • action—–设置数据发送的具体文件位置
    • method—–选择数据的提交方式（GET | POST）
    • GET—-传输不安全，传输量小2kb
    • POST—传输安全，传输量大
    • input—-单标签—输入框标签—收集用户数据
    • name—-定义数据名字
    • button—双标签—提交数据确认
    • type—–submit(默认值)
    语法结构：-----（重点）
                <form action="发送数据路径" method="GET">
                    用户名<input type="text" name="user"><br>
                    密码<input type="text" name="pwd"><br>
                    <button type="submit">登录</button>
                </form>

  3. input中常用的属性和type中常用的变化值

重点属性—–重点

value ----- 当input是输入框形态的时候设置其中输入的内容
                  当input是按钮状态的时候修改按钮显示的文字
                  placeholder----在输入框内提示输入内容，只要输入就不显示，不输入就显示
                  readonly----只读
                  disabled----禁用按钮
                  checked----默认选中

 type：
                  text(默认值)文本输入框
                  password----密码输入框
                  radio----单选框***必须使用相同的name才有单选的操作***
                  checkbox----多选框**变成按钮**
                  submit----提交按钮

了解：
                  button---普通按钮
                  reset--重置
                  file---文件上传
                  hidden---隐藏输入框

  1. 下拉选择框的组合操作和相关标签属性

    • 下拉选择框的的定义：通过下拉选择获取相关数据

                <select>
                    <option>请选择</option>
                    <option>中国</option>

                    <option>法国</option>
                </select>
